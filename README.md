## API: Generate a story

POST `/api/generate-story` with JSON:

```bash
curl -sS -X POST "http://localhost:3000/api/generate-story" \
  -H "Content-Type: application/json" \
  -d '{"teamName":"Platform","names":["Avery","Blake","Casey"]}'
```

Response:

```json
{
  "story": "..."
}
```

This endpoint requires `ANTHROPIC_API_KEY` to be set in your environment.

Optional: set `ANTHROPIC_MODEL` to override the default model (defaults to `claude-haiku-4-5`, with a fallback to `claude-3-5-haiku-20241022`).
